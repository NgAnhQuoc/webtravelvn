# Flatsome Expert Skill Set

- **UX Builder Mastery**: Ability to translate complex HTML structures into UX Builder rows, columns, and custom HTML elements.
- **Shortcode Development**: Proficiency in creating custom WordPress shortcodes that output consistent HTML for specific design components.
- **Flatsome Hook Integration**: Knowledge of `flatsome_after_header`, `flatsome_before_footer`, and other specific theme hooks.
- **Dynamic Data Mapping**: Skill in replacing static content with WordPress functions like `the_title()`, `the_content()`, and `get_field()` (ACF).
